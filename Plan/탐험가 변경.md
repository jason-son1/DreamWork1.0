# [개정안] 모험가 (Adventurer): 지평선의 기록자

## 1. 기본 설정 및 진행 방향 (Core Concept)

### 1.1. 기본 컨셉

- **초반 (Early Game):** 바닐라 생존과 동일하게 지도를 제작하고, 주변 지형을 밝히며, 배를 타고 이동하는 것으로 시작합니다.
    
- **중/후반 (Mid/Late Game):** 모험가 레벨과 미션(예: 정글 바이옴 발견, 폐광 탐사 등)을 달성함에 따라 **'전문 탐사가'**로 성장합니다.
    
- **핵심 변화:** 비현실적인 무빙(이단 점프 등) 대신, 이동 속도 증가, 낙하 피해 감소 같은 **'신체 능력 강화'**와, 좌표를 기록하고 공유하는 **'정밀 측량'** 능력에 집중합니다.
    
- **역할:** 직접 자원을 캐는 것보다, 희귀한 바이옴(산호초, 메사 등)이나 구조물(유적)의 **'위치(좌표)'**를 팔아 수익을 창출합니다.
    

### 1.2. 경험치 및 보상 테이블 (Config 예시)

모든 수치는 설정 파일에서 수정 가능합니다. 악용(매크로 이동)을 방지하기 위해 단순 이동보다는 '발견'에 비중을 둡니다.

|**대상 (Action)**|**등급 (Tier)**|**기본 경험치**|**기본 수입 (D)**|**비고**|
|---|---|---|---|---|
|**새로운 청크/지역 진입**|Normal|1.0|0.5|일정 시간/거리 쿨타임 적용|
|**희귀 바이옴 발견**|Rare|10.0|5.0|버섯섬, 메사, 정글 등 (서버 기록 기준 최초 진입 시 보너스)|
|**구조물 발견**|Epic|50.0|30.0|마을, 난파선, 유적 등 감지 시|
|**지도 제작**|Craft|5.0|2.0|지도 아이템을 꽉 채울 때|
|**몬스터 도감 등록**|Hunt|10.0|5.0|새로운 몹 최초 조우 시|

## 2. 스킬 (Skills) - 점진적 해금 시스템

복잡한 커맨드 스킬이나 비현실적인 액션을 삭제하고, 탐험을 쾌적하게 돕는 **패시브** 위주로 개편합니다.

### 2.1. [패시브] 험지 주파 (Pathfinder)

- **기능:** 특정 지형이나 블록 위에서 이동 속도가 빨라지고, 배고픔 소모가 줄어듭니다.
    
- **단계별 해금:**
    
    - **Lv. 1 ~ 9:** 효과 없음.
        
    - **Lv. 10+:** 흙길, 잔디 위에서 이동 속도 5% 증가.
        
    - **Lv. 30+:** 모래, 눈, 험한 지형 위에서 이동 속도 10% 증가 + 점프 강화 I 효과 미세 적용 (1블록을 매끄럽게 넘음).
        
    - **Lv. 50+:** 낙하 데미지 20% 영구 감소 + 높은 곳에서 떨어질 때 구르기(웅크리기) 시 데미지 무효화(쿨타임 적용).
        

### 2.2. [패시브] 육감 (Sixth Sense)

- **기능:** 주변에 숨겨진 구조물(던전, 폐광, 유적 등)이 있을 때 채팅창으로 희미한 힌트를 줍니다. (광부의 광맥 탐지와 유사)
    
- **효과:**
    
    - **Lv. 20:** 반경 30블록 내 몬스터 스포너 존재 여부 알림.
        
    - **Lv. 50:** 반경 50블록 내 희귀 구조물(엔드 유적, 사막 사원) 방향 힌트 제공.
        

### 2.3. [아이템 액티브] 귀환의 좌표 (Recall)

- **변경점:** 스킬 커맨드 삭제 → **'귀환석(Hearthstone)' 아이템** 사용으로 변경.
    
- **기능:** 모험가가 직접 제작한 귀환석을 사용하면 5~10초의 캐스팅(움직이면 취소) 후 타운(스폰)으로 이동합니다.
    
- **제약:** 레벨이 낮을수록 쿨타임이 길고(1시간), 레벨이 오르면 쿨타임이 줄어듭니다(10분).

## 3. 아이템 (Items)

PDC/NBT를 활용하여 기존 탐험 도구(나침반, 망원경, 지도)를 **'전문 장비'**로 업그레이드합니다.

### 3.1. 탐험 도구의 진화 (Tool Evolution)

#### A. 나침반 (Compass)

- **낡은 나침반:** 바닐라 나침반 (스폰 지점 표시).
    
- **탐지기 (Detector):** (Lv.20 제작)
    
    - 우클릭으로 모드 변경: 스폰 / 나의 죽은 위치 / 타운 방향.
        
- **바이옴 파인더 (Biome Seeker):** (Lv.50 제작 + 희귀 재료)
    
    - 가장 가까운 특정 바이옴(정글, 사막 등)의 방향을 가리킴. (쿨타임 및 내구도 존재)
        

#### B. 망원경 (Spyglass)

- **일반 망원경:** 바닐라 망원경.
    
- **이글 아이 (Eagle Eye):** (Lv.30 제작)
    
    - 줌인 상태에서 몬스터나 보물상자를 바라보면 **발광(Glowing)** 효과가 테두리에 표시됨. 벽 너머의 적을 정찰하는 용도.
        

#### C. 부츠 (Boots)

- **탐험가의 부츠 (Explorer's Boots):**
    
    - 가죽/사슬 부츠 기반. 방어력은 낮지만 **'이동 속도'**와 **'내구도'**가 매우 높음.
        
    - PDC를 통해 '착용 시 낙하 데미지 추가 감소' 옵션 부여.
        

### 3.2. 기록 및 판매용 아이템

#### A. 좌표 스크롤 (Coordinate Scroll)

- **기능:** 모험가가 빈 종이와 잉크를 들고 `/record` 하거나 제작대에서 조합하면, **현재 서 있는 위치의 좌표**가 기록된 아이템이 생성됩니다.
    
- **판매:** "메사 바이옴 좌표 팝니다" → 구매자가 아이템을 우클릭하면 해당 좌표가 채팅창에 뜨거나(기본), 나침반 타겟이 그곳으로 고정됩니다. (밸런스를 위해 즉시 텔레포트는 제한하거나 고레벨 전용으로 설정)
    

#### B. 바이옴 캡슐 (Biome Capsule)

- **기능:** 빈 유리병을 들고 특정 바이옴에서 상호작용하면 해당 바이옴의 **'공기'**가 담깁니다.
    
- **용도:** 타운 내 건축가나 주민들이 자기 땅의 잔디/나뭇잎 색깔을 바꾸고 싶을 때 소모품으로 사용합니다. (건축가들에게 고가에 판매)

## 4. 경제 및 타운 (Economy & Town)

### 4.1. 지도 제작소 (Cartography Office)

- **타운 NPC:** 지도 제작자.
    
- **탐사 보고 (Exploration Report):**
    
    - 모험가가 던전이나 유적을 발견하고 해당 위치에서 '보고서 작성(아이템 사용)'을 하면 **[탐사 보고서]** 아이템이 생성됩니다.
        
    - 이것을 NPC에게 납품하면 **경험치와 돈**을 받습니다. (구조물당 1회 제한 등으로 어뷰징 방지)
        
- **지도 판매:** 모험가가 제작한 '완성된 지도'를 납품하여 타운의 로비나 게시판에 지도를 장식할 수 있게 합니다.
    

### 4.2. 특송 배달 (Courier)

- 단순 퀘스트 형태. NPC가 **"이 소포를 야생 X:2000, Z:-5000 좌표에 있는 오두막에 배달해주게"**라고 의뢰합니다.
    
- 빠른 이동 속도를 가진 모험가가 제한 시간 내에 다녀오면 보상을 받습니다.

## 5. 이벤트 (Event)

**돌발 이벤트 삭제:**

- **에어드랍, 차원 균열 삭제:** 게임의 분위기를 해치는 아케이드성 요소를 제거합니다.
    
- 대신, **'최초 발견(First Discovery)'** 알림을 강화합니다.
    
    - _"모험가 000님이 서버 최초로 [고대 도시]를 발견했습니다!"_ (명예의 전당 기록)

## 6. 개발자 참고사항 (Implementation Note)

- **PDC/NBT 데이터:**
    
    - **나침반:** `target_biome: JUNGLE`, `target_x: 500` 등의 데이터를 저장하여, 플레이어의 나침반 바늘(`Player#setCompassTarget`)을 지속적으로 갱신해주는 스케줄러가 필요합니다.
        
    - **바이옴 캡슐:** `biome_type: DESERT` 데이터를 저장하고, 타운 플러그인(Towny/Lands)과 연동하여 해당 청크의 바이옴 값을 변경해주는 API 활용이 필요합니다.
        
- **좌표 기록 악용 방지:**
    
    - 좌표 스크롤 생성 시 `created_by: player_uuid`와 `timestamp`를 기록하여, 같은 위치의 좌표를 도배해서 파는 행위를 제제하거나 쿨타임을 둡니다.
        
- **Config 구조 (jobs/adventurer.yml):**
    
    YAML
    
    ```
    level-up-curve: "level * 100 + 50"
    skills:
      pathfinder:
        speed-boost-blocks: [GRASS_PATH, DIRT_PATH, SAND]
        speed-amount: 0.1 # 10% 증가
    items:
      biome_capsule:
        allowed-biomes: [CHERRY_GROVE, JUNGLE, BADLANDS]
        craft-level: 20
    rewards:
      structure-discovery:
        ANCIENT_CITY: 500.0 # 보상금
        VILLAGE: 50.0
    ```
