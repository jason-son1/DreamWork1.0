### 1. 경험치 및 보상 테이블 (Experience & Rewards)

### 2. 스킬 상세 명세 (Skill Mechanics)

액티브 스킬은 '귀환'과 '탐지' 도구 사용으로, 패시브는 이동 편의성으로 구현합니다.

#### 2.1. [패시브] 험지 주파 (Pathfinder)

- **기능:** 특정 블록 위에서 이동 속도 증가 및 배고픔 소모 감소.
    
- **구현 로직:** `PlayerMoveEvent`에서 `player.getLocation().getBlock().getRelative(BlockFace.DOWN).getType()` 확인.
    

|**레벨 (Lv)**|**적용 블록**|**효과**|**비고**|
|---|---|---|---|
|**10 ~ 29**|흙길(Grass Path), 잔디|속도 +10%|마을이나 평지 이동 시 유리.|
|**30 ~ 49**|모래, 눈, 자갈|속도 +15%|사막/설원 탐험 시 유리. 점프 강화 I 미세 적용.|
|**50 ~ Max**|**모든 지형**|속도 +20%|낙하 데미지 20% 영구 감소 및 구르기(Shift 착지) 가능.|

#### 2.2. [패시브] 육감 (Sixth Sense)

- **기능:** 주변의 숨겨진 구조물이나 위험 요소를 감지하여 채팅창 힌트 제공.
    
- **구현 로직:** 1~2분 간격 Scheduler 또는 새로운 청크 진입 시 `world.locateNearestStructure` API 호출 (서버 부하 고려 최적화 필수).
    

|**레벨 (Lv)**|**감지 대상**|**힌트 메시지 예시**|
|---|---|---|
|**20+**|몬스터 스포너, 폐광|"발 밑에서 둔탁한 소리가 들립니다..."|
|**50+**|엔드 유적, 고대 도시|"공기가 차가워집니다. 남동쪽에 고대 도시가 있습니다."|

#### 2.3. [아이템 액티브] 귀환의 좌표 (Recall)

- **아이템:** **귀환석 (Hearthstone)** (직접 제작).
    
- **사용법:** 우클릭 시 캐스팅(Casting) 바가 차오르며 타운(Spawn)으로 이동.
    
- **제약:** 이동하거나 피격 시 캐스팅 취소.
    

|**레벨 (Lv)**|**캐스팅 시간**|**쿨타임**|
|---|---|---|
|**초급 (Lv.10)**|10초|1시간|
|**중급 (Lv.30)**|7초|30분|
|**고급 (Lv.50)**|5초|10분|

### 3. 아이템 및 데이터 구조 (Items & PDC)

탐험가의 아이템은 **위치 정보(Coordinates)**와 **바이옴 데이터**를 저장하여 거래 가능한 형태로 만듭니다.

#### 3.1. 탐험 도구의 진화 (Tool Evolution)

- **기본 구조:** `Material.COMPASS`, `SPYGLASS` + PDC Data.
    

|**등급**|**아이템 명**|**제작/해금 조건**|**PDC/Lore 특징**|**특수 능력**|
|---|---|---|---|---|
|**탐지**|**탐지기 (Detector)**|Lv.20 + 철|`Mode: Spawn/Death`|우클릭으로 나침반 타겟 변경 (스폰 ↔ 최근 죽은 위치).|
|**추적**|**바이옴 파인더**|Lv.50 + 금|`Target: Jungle`|가장 가까운 **특정 바이옴** 방향을 가리킴 (쿨타임 존재).|
|**정찰**|**이글 아이 (Eagle Eye)**|Lv.30 (망원경)|`Ability: Glowing`|줌인(Zoom-in) 시 시야 내의 몬스터/상자 테두리 발광.|
|**장비**|**탐험가의 부츠**|가죽/사슬 기반|`Stat: Speed`|내구도가 매우 높고, 착용 시 이동속도/낙하보호 버프.|

#### 3.2. 기록 및 판매용 아이템 (Trade Goods)

1. **좌표 스크롤 (Coordinate Scroll)**
    
    - **생성:** 빈 종이 들고 `/record [이름]` 또는 제작대 조합.
        
    - **PDC:** `target_world`, `target_x`, `target_y`, `target_z`, `creator_uuid`.
        
    - **기능:** 구매자가 우클릭 시, 나침반이 해당 좌표를 가리키거나 채팅창에 좌표 출력.
        
    - **용도:** "메사 바이옴 좌표 팝니다", "엔드 유적 좌표 팝니다" 등 정보 거래.
        
2. **바이옴 캡슐 (Biome Capsule)**
    
    - **생성:** 빈 유리병을 들고 특정 바이옴에서 상호작용.
        
    - **PDC:** `biome_type: JUNGLE`, `biome_type: BADLANDS`.
        
    - **기능:** 타운 내 건축가가 사용 시, 해당 청크(Plot)의 바이옴 색상(잔디/나뭇잎 색)을 변경.
        
    - **가치:** 건축가들에게 매우 인기 있는 고가 아이템.

### 5. 경제 및 타운 시스템 연동 (Economy & Cartography)

탐험가는 물리적 자원이 아닌 **'정보(Information)'**를 팝니다.

#### 5.1. 지도 제작소 (Cartography Office)

- **NPC:** 지도 제작자.
    
- **탐사 보고 (Exploration Report):**
    
    - 유적 발견 시 `보고서 작성` 아이템 사용 → **[탐사 보고서]** 생성.
        
    - NPC에게 납품하면 대량의 경험치와 돈 지급. (구조물당 1회/서버 전체 1회 제한 등으로 밸런스 조절).
        
- **지도 판매:**
    
    - 탐험가가 직접 밝힌(Filled Map) 지도를 복제하여 액자용으로 판매.
        

#### 5.2. 특송 배달 (Courier Quest)

- **의뢰:** 타운 NPC가 주기적으로 배달 퀘스트 발주.
    
- **내용:** "이 소포를 야생 X:2000 좌표에 있는 오두막에 10분 내로 배달하시오."
    
- **보상:** 성공 시 골드 + 명성. 실패 시 위약금. 이동 속도가 빠른 탐험가에게 최적화된 돈벌이.
    

#### 5.3. 경제 순환 모델 (The Loop)

1. **탐험가**는 야생을 돌아다니며 **희귀 바이옴(메사/정글)**과 **유적**을 찾음.
    
2. 해당 위치를 **좌표 스크롤**로 만들어 **광부(자원 채취)**나 **사냥꾼(던전 공략)**에게 판매.
    
3. 아름다운 바이옴의 공기를 **바이옴 캡슐**에 담아 **건축가(Town Builder)**에게 판매.
    
4. 획득한 돈으로 내구도가 높은 **탐험가의 부츠**나 **귀환석 재료**를 구매.

### 💡 개발자(User)를 위한 구현 팁 (Technical Note)

1. **Event Listener:**
    
    - `PlayerMoveEvent`:
        
        - **험지 주파:** `getTo().getBlock().getType()` 체크. 매 틱마다 체크하면 무거우므로, `from.getBlock() != to.getBlock()`일 때만 로직 실행.
            
        - **육감/바이옴 발견:** 비동기 태스크(`runTaskAsynchronously`)로 처리하여 서버 렉 방지 필수.
            
    - `PlayerInteractEvent`:
        
        - **좌표 스크롤:** 우클릭 시 `player.setCompassTarget(targetLoc)` 실행 및 메시지 출력.
            
        - **바이옴 캡슐:** `Towny API` 또는 `WorldEdit API`를 사용하여 해당 청크의 `setBiome` 실행.
            
2. **Config 구성 (`jobs/adventurer.yml`):**
    
    YAML
    
    ```
    skills:
      pathfinder:
        blocks: [GRASS_PATH, DIRT_PATH, SAND, GRAVEL]
        speed-boost: 0.1 # WalkSpeed +0.1
      recall:
        cooldown: [3600, 1800, 600] # 초 단위 (Lv 10, 30, 50)
    
    rewards:
      discovery:
        JUNGLE: 500 # 최초 발견 시 추가 보상금
        ANCIENT_CITY: 1000
    ```
    
3. **PDC & NBT:**
    
    - 좌표 스크롤 생성 로직:
        
        Java
        
        ```
        // Pseudo Code
        ItemStack scroll = new ItemStack(Material.PAPER);
        ItemMeta meta = scroll.getItemMeta();
        meta.getPersistentDataContainer().set(keyX, INTEGER, loc.getBlockX());
        meta.getPersistentDataContainer().set(keyZ, INTEGER, loc.getBlockZ());
        meta.setDisplayName("§b[좌표 스크롤] " + biomeName);
        scroll.setItemMeta(meta);
        ```
        
    - 이렇게 저장된 데이터는 서버가 재시작되어도 유지되며, 다른 플러그인(상점 등)과 연동하기 쉽습니다1111.
