# [개정안] 농부 (Farmer): 대지의 관리자 및 미식가

## 1. 기본 설정 및 진행 방향 (Core Concept)

### 1.1 기본 컨셉

- **초반 (Early Game):** 바닐라 마인크래프트와 동일. 괭이로 땅을 갈고, 씨앗을 심고, 물을 긷는 노동 집약적 농사.
    
- **중반 (Mid Game):** 농부 레벨이 오르고 특정 미션을 달성함에 따라 '기능성 도구'와 '비료'를 사용하여 효율 증대.
    
- **후반 (Late Game):** 자동 재파종, 범위 수확, 성장 촉진 등이 아이템과 패시브 스킬로 자동화됨. 고품질 요리와 술을 타운에 납품하여 고수익 창출.
    
- **시스템 원칙:** 모든 확률, 경험치, 해금 레벨, 아이템 능력치는 자체 플러그인(`DreamWork`)의 Config 파일에서 운영자가 직접 수정 가능해야 함.
    

### 1.2 경험치 및 보상 테이블 (Config 예시)

- **기본 원칙:** 단순 설치/파괴 반복 악용을 막기 위해, **'완전히 자란 작물(Fully Grown)'**을 수확할 때만 경험치와 재화 지급.
    

|                 |               |            |               |              |
| --------------- | ------------- | ---------- | ------------- | ------------ |
| **대상 (Action)** | **등급 (Tier)** | **기본 경험치** | **기본 수입 (D)** | **비고**       |
| 밀/감자/당근         | Tier 1        | 1.0        | 0.5           | 기초 작물        |
| 호박/수박           | Tier 1        | 1.5        | 0.8           | 자동화 농장 효율 고려 |
| 사탕수수/선인장        | Tier 1        | 1.0        | 0.5           | 성장 속도 빠름     |
| 네더와트/코코아        | Tier 2        | 3.0        | 2.0           | 특정 환경 필요     |
| 황금 작물           | Tier 3        | 50.0       | 100.0         | 확률적 대박 보상    |

## 2. 스킬 (Skills) - 점진적 해금 시스템

스킬을 처음부터 지급하지 않고, **레벨 달성** 및 **진급 심사(미션)**를 통해 단계적으로 해금합니다.

### 2.1 [패시브] 녹색 손길 (Green Thumb)

- **기능:** 작물 수확 시 자동으로 인벤토리의 씨앗을 소모해 다시 심음.
    
- **단계별 해금 (Configurable):**
    
    - **Lv. 1 ~ 9:** 효과 없음 (수동 농사).
        
    - **Lv. 10 + [미션: 밀 1,000개 수확]:** 50% 확률로 재파종 발동.
        
    - **Lv. 30 + [미션: 호박 500개 납품]:** 100% 확률로 재파종 발동.
        

### 2.2 [패시브] 대지의 기운 (Growth Aura)

- **기능:** 농부 주변의 작물 성장 속도가 빨라짐.
    
- **차등 효과:**
    
    - **Lv. 1:** 효과 없음.
        
    - **Lv. 20:** 반경 3블록 내 성장 속도 5% 증가.
        
    - **Lv. 50:** 반경 5블록 내 성장 속도 10% 증가.
        
    - **Lv. MAX:** 반경 7블록 내 성장 속도 20% 증가 + 주변 작물이 마르지 않음(물 불필요).
        

### 2.3 [아이템 액티브] 풍년의 의식 (Rite of Harvest)

- **변경점:** 쿨타임 스킬(커맨드) 방식 삭제 → **'농부 전용 소모품/도구'** 사용 방식으로 변경.
    
- **기능:** 아이템(예: 풍요의 뿔피리) 사용 시 주변 작물 즉시 성장.
    
- **아이템 등급별 효과:**
    
    - **하급 뿔피리:** 반경 3x3 작물에 뼛가루 효과 1회. (소모성)
        
    - **상급 뿔피리:** 반경 9x9 작물을 즉시 최대 성장 단계로 만듦. (내구도 소모형)
        
    - _이 아이템들은 농부 레벨이 높아야 제작하거나 효율적으로 사용 가능._
        

## 3. 아이템 (Items)

- **구현 방식:**
    
    - **1순위:** Oraxen / ItemsAdder 플러그인 활용 (커스텀 텍스처 적용).
        
    - **2순위:** 자체 플러그인(`DreamWork`)의 PDC(PersistentDataContainer)를 활용하여 바닐라 아이템에 커스텀 데이터 부여.
        

### 3.1 황금 씨앗 (Golden Seed)

- **획득 조건:** 오직 **'완전히 자란 작물'**을 직접 수확할 때만 낮은 확률로 드롭. (잡초 제거, 덜 자란 작물 파괴 시 드롭 X)
    
- **확률 공식:** `BaseChance + (FarmerLevel * 0.05%)`
    
    - 농부 레벨이 높을수록 더 자주 얻습니다.
        
- **용도:** 심으면 랜덤한 보상(다이아몬드, 인챈트북, 특수 비료 등)이 열리는 '황금 작물' 생성.
    

### 3.2 다양한 비료 시스템 (Fertilizers)

비료의 종류를 세분화하여 농사의 전략성을 더합니다.

1. **속성 비료 (Quick-Gro):** 성장 속도 2배 가속 (지속 시간 10분).
    
2. **품질 비료 (Quality Compost):** 수확 시 '2성(★★)' 이상의 작물이 나올 확률 대폭 증가.
    
3. **변이 비료 (Mutation Mix):** 인접한 작물 교배 시 희귀 변종(예: 황금 당근 등) 발생 확률 증가.
    
4. **보습 비료 (Hydro Gel):** 물이 없는 곳에서도 작물이 자라게 하며, 경작지가 짓밟혀도 보호됨.
    

### 3.3 장비: 괭이의 진화 (Hoe Evolution)

광부의 곡괭이처럼, 괭이도 성장합니다. (초반엔 철 괭이 사용 권장)

- **초심자의 괭이 (Starter Hoe):** 내구도만 약간 높은 철 괭이.
    
- **개량된 괭이 (Improved Hoe):**
    
    - 조건: 농부 Lv.20 이상 제작 가능.
        
    - 효과: 수확 시 10% 확률로 수확물 2배.
        
- **대지주의 괭이 (Landlord's Hoe):**
    
    - 조건: 농부 Lv.50 이상.
        
    - 효과: **[범위 수확]** 우클릭 시 3x3 범위 경작/수확. **[자동 심기]** 기능 내장.
        
- **커스텀 인챈트:** 괭이 전용 인챈트 (예: `Wide Tilling` - 경작 범위 증가, `Bountiful` - 추가 드롭 등) 추가 가능.
    

## 4. 경제 및 타운 (Economy & Processing)

### 4.1 작물 등급 시스템 (Crop Quality)

단순 농사가 아닌 '품질 관리' 개념 도입.

- **1성 (★):** 일반 작물. (기본 가격)
    
- **2성 (★★):** "싱싱한" 접두사. 상점에 1.5배 가격으로 판매 가능.
    
- **3성 (★★★):** "유기농/황금" 접두사. 요리 및 양조의 필수 재료. 상점 판매 불가(오직 가공용).
    
- **구현:** 농부 레벨이 높을수록 고등급 수확 확률 증가. (Config에서 확률 조정)
    

### 4.2 요리 및 양조 (Cooking & Brewing)

레스토랑 경영(타이쿤) 요소는 삭제하고, **'제작 및 납품'**에 집중합니다.

- **요리:**
    
    - 타운의 공용 '주방' 혹은 개인 설치 가구에서 제작.
        
    - 농부 레벨에 따라 레시피 해금 (예: Lv.10 샌드위치 -> Lv.50 스테이크 정식).
        
    - **효과:** 배고픔 회복을 넘어선 **RPG 버프** (체력 증가, 채광 속도 증가 등) 제공.
        
- **양조 (주조):**
    
    - 밀(맥주), 포도/베리(와인), 감자(보드카) 등을 오크통에 숙성.
        
    - 숙성 시간이 길수록(실제 시간 비례) 등급 상승.
        
    - 고등급 술은 타운의 **NPC(주점 주인)**에게 매우 비싸게 납품 가능.
        

### 4.3 경제 구조

1. **1차 수익:** 수확한 일반 작물(1~2성)을 타운의 '농산물 거래소 NPC'에게 판매.
    
2. **2차 수익:** 3성 작물로 '버프 요리'를 만들어 사냥꾼/광부 유저에게 판매 (유저 간 거래).
    
3. **납품 퀘스트:** 타운 NPC가 주기적으로 "오늘은 호박 파이 50개가 필요해" 같은 의뢰를 주며, 농부 레벨에 따라 보상 차등 지급.
    

## 5. 이벤트 및 기타

- **돌발 이벤트 삭제:** 농부 유저가 편안하게 농사에 집중할 수 있도록 메뚜기 떼, 기상 악화 등의 스트레스성 랜덤 이벤트는 모두 제거합니다.
    
- **계절 시스템 (선택 사항):** 가능하다면 계절(봄/여름/가을/겨울)에 따라 잘 자라는 작물이 달라지도록 설정하여 작물의 다양성을 유도 (Config에서 On/Off).
    

## 6. 개발자 참고사항 (Implementation Note)

- **Config 구조 제안 (`jobs/farmer.yml`):**
    
    ```
    level-up-curve: "level * 100 + 50" # 레벨업 필요 경험치 공식
    
    skills:
      green-thumb:
        unlock-level: 10
        chance: 50 # %
        conditions:
          - "mission:harvest_wheat_1000"
      growth-aura:
        radius-per-rank: [0, 3, 5, 7] # 레벨 구간별 반경
    
    items:
      golden-seed:
        base-chance: 0.1 # 기본 드롭률 0.1%
        bonus-per-level: 0.05 # 레벨당 추가 확률
        drop-from: [WHEAT, CARROT, POTATO] # 드롭 대상 작물
        prevent-exploit: true # 블록의 Age 데이터 확인 필수 (다 자란 것만)
    ```
    
- **PDC 활용:** 수확한 작물 아이템에 `quality: 2` 같은 NBT 태그를 붙여 등급을 구분하고, 상점 플러그인이나 요리 시스템에서 이 태그를 인식하도록 개발해야 함.
